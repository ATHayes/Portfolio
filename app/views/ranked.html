Ranked.ie

Ranked.ie is an online forum where members can add, read, rate and comment on articles. Each article has a topic. A user can write multiple articles and make multiple comments. Users can also be curators. Complaints about content are made to curators, who may remove content as necessary.

Create a data model for this system. Then create a Microsoft Access database to demonstrate that data model.

A suitable database model (Entity Relationship Diagram) was designed using UML, with the following assumptions:

  • An article can only be written under one topic
  • Curators can only handle one topic
